System Documentation

Introduction

Welcome to the AI-based ETH Blockchain Security Detection System, a comprehensive platform designed to assist users in monitoring and analyzing smart contracts and transactions for potential security threats. This documentation will guide you through the various features and functionalities of our platform, ensuring a streamlined experience in detecting and managing blockchain fraud and vulnerabilities.

Getting Started

Before using the system, ensure that you have a compatible web browser and a stable internet connection. For optimal performance, we recommend using the latest version of Chrome or Firefox.

Dashboard Overview

The dashboard is your central hub for monitoring and analyzing activities. It provides a quick overview of recent smart contract analyses and fraud transaction detections.

Features:

  • Real-time data visualization
  • Access to new analyses and historical data
  • Quick navigation to all system functions

Smart Contracts

The platform allows for the in-depth analysis of smart contracts using the Solidity language.

Analyzing New Contracts:

  • Navigate to 'Analysis New Contract'.
  • Enter the smart contract's name, Solidity version, and code into the respective fields.
  • Click 'Analyze Contract' to begin the process.

Reviewing Analysis History:

  • Click on 'Smart Contract Analysis History' to view past analyses.
  • Utilize the search function to find specific entries by contract name or vulnerability type.
  • Select an entry to view detailed analysis results.

Fraud Transaction Detection

This module enables users to analyze transactions on the Ethereum blockchain for fraudulent activity.

Submitting Transactions Hash for Analysis:

  • Go to 'Analysis New Transactions'.
  • Input the transaction name and transaction hash.
  • Click 'Analyze Transaction' to receive the risk assessment report.

Fraud Transaction Analysis History:

  • Access past analyses under 'Fraud Transaction Analysis History'.
  • Search by transaction name, fraud type, or date for ease of access.
  • Select an entry to view detailed analysis results.
